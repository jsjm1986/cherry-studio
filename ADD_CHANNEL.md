# 主机频道使用指南

## 这是什么
主机频道让你把多名“专家”拉进同一个房间，用协作方式和 AI 对话。主机负责房间配置，专家按各自身份参与，话题记录在房间下，支持继续聊、重命名或删除。

## 入口
- 侧边栏/顶栏：图标“主机”（👥）。若没看到，可在侧边栏设置中启用。
- 标签页模式：点击“主机”标签进入。
- Launchpad：主页快捷卡片“主机”。

## 核心概念
- 主机（房间）：一个协作会话容器，包含提示词/模型/工具等设置，可有多个话题。
- 专家：绑定到主机的角色，拥有独立的名字、头像、提示词和工具/模型设置，可被 @。
- 话题：主机内的对话线程，支持重命名、切换和删除。

## 快速上手
1) 创建主机：进入主机页 → 左侧“创建房间”或“+” → 选表情、命名、写一句房间描述/背景。
2) 添加专家：在“专家”区域点击“+”或“导入助手”。
   - 自建：填头像、名称、@handle、关键词、风格提示词。
   - 导入：从现有助手批量勾选导入为专家。
3) 开聊：
   - 选中房间后，右侧聊天区输入即可。
   - 选专家：输入框上方点击“@ 选择专家”，或在专家列表点击“@”按钮。
   - 话题：右侧顶部“新建对话”创建；点更多可重命名或删除。

## 界面速览
- 左侧栏：当前房间信息 → 房间列表（…菜单可改名/删房间）→ 专家列表（…菜单可改/删，@ 按钮快速插入）→ 话题列表（可新建/重命名/删除）。
- 右侧：聊天区（沿用主聊天能力，支持 Quick Panel、选择文本工具条）+ 输入框（显示当前选中专家标签，可一键清除）。

## 工具 / 知识库 / 模型
- 每个专家都能独立配置模型、知识库、MCP 服务器、Web 搜索开关与 Provider（专家列表更多 → “设置”里的“模型 / 知识库 / MCP”标签）。
- 主机也可配置这些能力；专家未单独设置时自动继承主机配置，避免重复填。
- 发送时会合并配置：优先用专家的，缺省项用主机兜底；默认工具调用模式为 function call，可在群组里使用 Web 搜索、MCP 工具、知识库检索、记忆、图片生成等。
- 知识库检索、MCP 调用、Web 搜索 Provider 的选择都按“专家优先、主机兜底”策略执行。

## 专家&房间设置要点
- 提示词增强：专家默认开启“提示词增强模式”，会自动加身份强化，保持角色一致性。
- 主机提示词融合：专家设置里可选择“附加主机提示词”或“忽略主机提示词”。
- 模型/工具：在专家设置弹窗的“模型 / 知识库 / MCP”标签可单独配置，未设置时继承主机。
- 话题共享：专家不单独存话题，使用主机的话题线程。

## 常见操作
- 切换话题：左侧话题列表点击即可，最近更新排在前面。
- 重命名话题：话题项右侧更多 → 重命名。
- 删除话题：话题项右侧更多 → 删除（不可恢复）。
- 编辑房间：房间列表项右侧“铅笔”。
- 删除房间：房间列表项右侧“删除”（会同时移除该房间的专家；话题/消息的彻底清理按应用后端设计执行）。
- 调出专家设置：专家列表的更多 → “设置”，可调模型/知识库/MCP、提示词增强策略。

## 提示与限制
- 数据清理：删除房间/专家主要移除前端记录；若需彻底清理历史消息，请根据团队数据策略执行（当前版本未自动清空关联话题消息）。
- 话题初始化：房间创建后首次进入会生成默认话题；若从未进入过房间可能没有写入本地数据库，需进入一次即可。
- 多语言：部分文案在非中/英语言下可能显示占位文本，属待翻译项。
- 文字选中工具条：复制可用，其余功能为预留项，后续版本完善。

## 遇到问题？
- 看不到主机图标：检查侧边栏可见图标或在设置中启用。
- 无专家可选：先在左侧添加或从助手导入。
- 提示词未生效：在专家设置中确认“提示词增强模式”与“主机提示词”策略；确保选中了对应专家。
